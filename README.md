# Churn_prediction_case_study
Using ML models to predict and reduce churn in the ride-sharing industry.
From a thorough EDA, through validating and assessing different model performances, to creating data-driven recommendations.

Churn reduction is a challenge that most service based companies are facing when growing alongside fierce competition.
This case study will present a method to explore and analyze some root causes based on a mock dataset and suggest solutions to deal with churn reduction using python.
